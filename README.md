# Susy Snippets

This is a collection of Sublime Text snippets for [Susy][1]. These will work on files with the `.scss` extension for now.

Here are the tab triggers in alphabetical order:

* `al` - Alpha Mixin
* `bl` - Bleed Mixin
* `blx` - Bleed-X Mixin
* `bly` - Bleed-Y Mixin
* `br` - Break Mixin
* `cof` - Container Function
* `co` - Container Mixin
* `fi` - First Mixin
* `fu` - Full Mixin
* `ga` - Gallery Mixin
* `gbs` - Global Box Sizing Mixin
* `guf` - Gutters Function
* `gu` - Gutters Mixin
* `is` - Isolate Mixin
* `la` - Last Mixin
* `lo` - Layout Mixin
* `nef` - Nested Function
* `ne` - Nested Mixin
* `nb` - Nobreak Mixin
* `om` - Omega Mixin
* `pad` - Pad Mixin
* `po` - Post Mixin
* `pr` - Pre Mixin
* `px` - Prefix Mixin
* `pl` - Pull Mixin
* `ph` - Push Mixin
* `sg` - Show Grid Mixin
* `spf` - Span Function
* `sp` - Span Mixin
* `sq` - Squish Mixin
* `sx` - Suffix Mixin
* `wl` - With Layout Mixin

[1]:[http://susy.oddbird.net/]
